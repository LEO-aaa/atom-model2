# 原子结构模型 - GitHub Pages 版

这是一个用于物理课堂演示的 **3D 原子结构模型**，可以在浏览器中拖动旋转视角、缩放远近，适配 iPad Safari。

## 使用方式

1. 在 GitHub 创建一个新的仓库，例如：`atom-model-demo`
2. 将本项目中的 **`index.html`** 上传到该仓库（可以直接在网页端新建文件并粘贴内容）。
3. 在 GitHub 仓库中打开 **Settings → Pages**：
   - `Source` 选择：**Deploy from a branch**
   - `Branch` 选择：`main` 分支，目录 `root`，然后保存。
4. 等待 GitHub Pages 构建完成后，页面上会显示你的网站地址，例如：

   `https://你的用户名.github.io/atom-model-demo/`

5. 在 iPad 上用 Safari 打开这个地址即可课堂演示。
